Getting Started

Fastmail TUI is a terminal-based email client designed for privacy, speed, and efficiency. Here's how to get started:

1

Activate the Environment

Open your terminal and activate the virtual environment:

source ~/Documents/GitHub/fastmail-tui/venv/bin/activate
2

Run Setup

Configure your Fastmail API token and optional Claude API key:

fm setup

Get your Fastmail API token from: Settings > Privacy & Security > Integrations > API Tokens

3

Launch the TUI

Start the email client:

fm

Or use the full command: fastmail-tui

Terminal - Fastmail TUI
$ fm

Connecting to Fastmail...
Loaded 127 emails from Inbox
AI assistant ready
Press ? for help

Configuration

Customize Fastmail TUI to your preferences.

Config File Location

~/.config/fastmail-tui/config.yaml

Default Configuration

fastmail:
  host: api.fastmail.com

claude:
  enabled: true
  model: claude-sonnet-4-5

cache:
  enabled: true
  max_messages: 500

ui:
  vim_mode: true
  show_ai_panel: true
  refresh_interval: 30
  page_size: 50

CLI Configuration Commands

# Enable/disable AI
fm configure --enable-ai
fm configure --disable-ai

# Set refresh interval (seconds)
fm configure --refresh-interval 60

# Show config file path
fm config-path

# Clear all stored credentials
fm clear-credentials
